About Netflux2


Version alpha2 (3/30/2025), under development. To provide feedback, see Help.

Netflux2 is a user-friendly tool for simulating biological networks. The best way to get started with Netflux is using the tutorial. The tutorial is accepted at PLOS Computational Biology and the preprint is here: bioRxiv

Netflux simulates networks using a modeling approach called logic-based differential equations, described here: Kraeutler MJ, Soltis AR, Saucerman JJ. Modeling cardiac β-adrenergic signaling with normalized-Hill differential equations: comparison with a biochemical model. BMC Syst Biol. 2010 Nov 18;4:157. link (open access)

The source code for Netflux2 is available on GitHub under an MIT open source license.

The previous version Netflux (v1) currently is better tested and has more features. It is based in MATLAB and also has a Windows installer for those without MATLAB. The source code is available on GitHub under an MIT open source license.
