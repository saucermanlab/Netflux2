About Lorenz


This Lorenz web app allows for interactive simulation of the Lorenz system. The Lorenz system was developed by Edward Lorenz, initially as a model of the atmosphere. Lorenz discovered that this system has chaotic behavior, which gave rise to the concept of the "butterfly effect", asking "Does the flap of a butterfly's wings in Brazil set off a tornado in Texas?" link

For more information about the Lorenz equations, see: Lorenz system (Wikipedia)

This Lorenz web app was developed by Jeff Saucerman using Python and Flask.

The source code is available on GitHub under an MIT open source license.
