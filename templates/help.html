Help


Netflux2(version alpha2, 3/30/2025) is under development. We appreciate your feedback by submitting this brief survey.

You are also welcome to email Jeff Saucerman at jsaucerman@virginia.edu if you find bugs or have suggestions.

The Netflux documentation and tutorial is accepted at PLOS Computational Biology, with preprint available here.

Video tutorials are available on YouTube.

Running basic simulations

Netflux2 automatically loads exampleNet. You can then run basic simulations using the below buttons and parameters.

Time span: specify the duration of the simulation.
Simulate: click to run a simulation with the current parameters.
Plot: replot the simulation using only the specified species.
Reset parameters: reset the species and reaction parameters back to their original values.
Reset simulation: reset time and the timecourse of species activities.

Species parameters

Species: select the species that you would like to adjust.
Y0: initial value for that species. By default, we use Y0 = 0.
Ymax: maximum value for that species. By default, we use Ymax = 1.
tau: time constant that determines how fast that species can change. By default, we use tau = 0.1 h for signaling, 1 h for transcription, and 10 h for translation.

Reaction parameters

Reactions: select the reaction that you would like to adjust.
w: reaction weight for the strength of reaction, 0 <= w <= 1. By default, we use w = 1.
n: Hill coefficient, affecting apparent cooperativitiy of activation. By default, we use n = 1.4.
EC50: sensitivity of the reaction to the upstream reactant. By default, we use EC50 = 0.5, so that half-maximal reactant causes half-maximal activation of the product.

The above default parameter values were determined in Kraeutler et al. BMC Sys Bio 2010. In that study, we optimized the default parameters to emulate the behavior of a validated mass-action kinetic network of beta-adrenergic signaling (Saucerman et al, JBC 2003; Saucerman et al, PBMB 2004).

Menu items

Open Model: open Netflux model file in XLSX format.
Download Simulation: download timecouse of species activities in CSV format.
Download as Python: download Python files modelName_run.py, modelName_ODEs.py, and modelName_params.py. This allows you to perform more advanced simulations directly in Python.
Download as XGMML: download network visualization in XGMML format, which can by loaded by Cytoscape or other packages.
Help: this page.
